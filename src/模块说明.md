# LightC 前端模块说明

> 基于 React + TypeScript + Tailwind CSS，Tauri v2 桌面端

---

## 目录结构

```
src/
├── api/                # API 层：Tauri invoke 封装 + 类型定义
├── components/         # UI 组件
│   ├── modules/        #   业务模块（10 个功能模块）
│   └── *.tsx           #   通用 UI 组件
├── contexts/           # React Context 全局状态
├── hooks/              # 自定义 Hooks
├── utils/              # 工具函数
├── App.css             # 全局样式 + CSS 变量
├── App.tsx             # 根组件
└── main.tsx            # 入口
```

## 二、业务模块 — [components/modules/](components/modules/)

### 大目录分析 — [HotspotModule.tsx](components/modules/HotspotModule.tsx)
**C 盘空间热点扫描**。支持浅扫描（AppData，秒级）和深度扫描（全 C 盘）。Rayon 多线程并行后端，智能下钻（>5GB 目录自动展开子目录）。实时进度条 + 取消支持（v2.3）。

### 下钻模态框 — [DrillDownModal.tsx](components/modules/DrillDownModal.tsx)
**沉浸式目录下钻弹窗**。Portal 渲染、面包屑导航、ESC 关闭、无限层级下钻。复用 `scanPathDirect` 后端（Rayon 并行）。支持清理回调同步到父组件。

**v2.3 性能优化**：
- `React.memo` 包裹条目组件（避免父组件 state 变化波及已渲染条目）
- 默认展示 30 条上限，超出显示「展示全部」按钮（防止大目录数百子目录撑爆 DOM）
- `useMemo` 缓存面包屑计算（避免每次 render 重复 split/循环）

### 垃圾清理 — [JunkCleanModule.tsx](components/modules/JunkCleanModule.tsx)
**系统垃圾文件智能扫描与清理**。多类别（Windows 缓存、浏览器缓存、日志文件、临时文件等）。分类折叠展示 + 全选/单选删除。

### 大文件扫描 — [BigFilesModule.tsx](components/modules/BigFilesModule.tsx)
**C 盘大文件扫描 (Top-N)**。实时进度推送（`large-file-scan:progress`）。风险分级标记（1-5 级）+ 多选批量删除。支持取消扫描。

### 社交软件专清 — [SocialCleanModule.tsx](components/modules/SocialCleanModule.tsx)
**8 款社交软件缓存专项清理**（微信/QQ/NTQQ/钉钉/飞书/Lark/企业微信/Telegram）。五规则优先级分类、增量勾选。

### 系统瘦身 — [SystemSlimModule.tsx](components/modules/SystemSlimModule.tsx)
**Windows 系统瘦身三件套**：休眠文件、虚拟内存、WinSxS 组件存储。管理员权限检测、实时 DISM 进度推送。

### 卸载残留 — [LeftoversModule.tsx](components/modules/LeftoversModule.tsx)
**卸载残留扫描与清理**。置信度评分展示、双删除模式（普通删除含可执行文件预检查 + 深度清理 3 步确认）。模拟器/虚拟磁盘残留检测、分类标签（高置信度/可疑）、「选择可疑项」批量操作。

### 注册表清理 — [RegistryModule.tsx](components/modules/RegistryModule.tsx)
**注册表残留扫描与清理**。HKCR\Applications 关联检测。备份文件生成 + 恢复引导。

### 右键菜单清理 — [ContextMenuModule.tsx](components/modules/ContextMenuModule.tsx)
**Windows 右键菜单失效项清理**。HKCU/HKLM 分层权限、MUIVerb 间接字符串解析、三级风险徽标（safe/caution/danger）、ContextMenuHandlers 系统保护禁止选中、.reg 备份自动导出。

### C 盘全盘分析 — [DiskGrowthModule.tsx](components/modules/DiskGrowthModule.tsx)
**C 盘空间变化分析**。通过 MFT 扫描建立目录快照，与上次扫描结果对比新增/减少空间，并展示对应变化目录；这里只做定位分析，不提供直接清理入口。
